About this dashboard

This is a dashboard of eBird data for Maharashtra. The original version (divided into seasons) was analysed first.

You'll notice that the Monthly (automated) and Seasonal (automated) views have more sparse analysis, and that's because they only really exist because I wanted to delve into CI/CD with GitHub Actions & YAML workflows, so I was focused on learning THAT and encountered some difficulty in automating some parts of my original process. For example, Habitat specializations, Migratory patterns, and Common behaviours were taken from stateofindiasbirds.in/. I could've done some stuff with the Python scripts to repeat the process of adding that data to the monthly data, but frankly I got tired and wanted to just move onto the CI/CD stuff and focus on learning how the API call works and to future projects.

I may improve this dashboard, but it was mostly a prototype so I could figure out a template for displaying a map and some extra data on the side, and also so I could learn CI/CD stuff (intention to use it in future projects). Again, I wanted to move on.

Did the entirety of the original analysis by myself, used ChatGPT to do some parts of the automation process (along with cleaning up my newbie attempts at the webdev stuff).

I will be using the knowledge gained in making this to create an SDM dashboard later, where the right-side window will have a NICE looking dashboard. Once I've deployed it, a link will be available here.
